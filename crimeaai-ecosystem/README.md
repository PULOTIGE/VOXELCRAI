## 🌟 Особенности

### Биологические структуры данных

- **🧬 Nucleotide (256 байт)** — базовая ячейка памяти с эпигенетическими метками, квантовым шумом и семантическим вектором
- **🌍 Voxel (9 КБ)** — микро-организм с сенсорами, физикой, мыслями, эмоциями и памятью
- **💡 LightPattern (1 КБ)** — паттерн освещения со сферическими гармониками

### Эмоциональная модель

- **⚡ KaifEngine** — движок «кайфа» на основе производной энтропии |dS/dt|
- 8 базовых эмоций: радость, грусть, гнев, страх, удивление, отвращение, любопытство, покой
- Адаптивный контроллер для поддержания оптимального состояния

### Поиск концептов

- 🔍 Интеграция с DuckDuckGo для поиска новых знаний
- Автоматическое извлечение терминов из текста
- Интеграция концептов в память вокселей

### Визуализация

- 🎨 Красивый cyberpunk UI на pygame
- Визуализация вокселей как светящихся точек
- ДНК-подобная визуализация нуклеотидов
- Колесо эмоций и графики в реальном времени

## 📦 Установка

### Требования

- Python 3.10+
- pip

### Установка зависимостей

```bash
cd crimeaai-ecosystem
pip install -r requirements.txt
```

### Запуск

```bash
# С графическим интерфейсом
python main.py

# Без GUI (консольный режим)
python main.py --no-gui

# Запуск тестов
python main.py --test
```

## 🔌 Плагины

Система поддерживает плагины для расширения функциональности.

### Создание плагина

```python
# plugins/my_plugin.py
from utils.plugins import PluginBase, PluginInfo

class MyPlugin(PluginBase):
    @property
    def info(self) -> PluginInfo:
        return PluginInfo(
            name="My Plugin",
            version="1.0.0",
            author="You",
            description="Описание плагина",
            dependencies=[]
        )
    
    def initialize(self, ecosystem):
        print("Plugin initialized!")
        return True
    
    def shutdown(self):
        print("Plugin shutdown")
    
    def on_tick(self, dt: float):
        # Вызывается каждый тик
        pass
```

## 📊 Структуры данных

### Nucleotide (256 байт)

```
┌──────────────────────────────────────────────┐
│ base (1 байт)           │ A, T, G, C         │
│ epigenetic_tags (7 байт)│ Метки модификаций  │
│ quantum_noise (4 байта) │ Псевдорандом       │
│ histone_state (16 байт) │ Состояние хроматина│
│ semantic_vector (228 б) │ Семантика (512→114)│
└──────────────────────────────────────────────┘
```

### Voxel (9 КБ)

```
┌──────────────────────────────────────────────┐
│ metadata    │ 512 Б  │ ID, позиция, здоровье │
│ sensors     │ 1.5 КБ │ Визуальные, аудио... │
│ physics     │ 1 КБ   │ Силы, коллизии       │
│ thoughts    │ 2 КБ   │ Фокус, рабочая память│
│ emotions    │ 2 КБ   │ 8 базовых эмоций     │
│ memory      │ 2 КБ   │ Долговременная память│
└──────────────────────────────────────────────┘
```

### LightPattern (1 КБ)

```
┌──────────────────────────────────────────────┐
│ pattern_id  │ 4 байта │ ID паттерна         │
│ direct      │ 384 Б   │ 32 источника x RGB  │
│ indirect    │ 384 Б   │ Отражённое освещение│
│ sh_coeffs   │ 108 Б   │ 9 SH коэффициентов  │
│ material    │ 40 Б    │ roughness, metalness│
└──────────────────────────────────────────────┘
```

## ⚡ Kaif (Кайф)

**Кайф** — это абсолютная величина производной энтропии системы:

```
KAIF = |dS/dt|
```

где S — энтропия Шеннона эмоционального состояния.

### Состояния кайфа:

| Состояние | Диапазон | Описание |
|-----------|----------|----------|
| `dormant` | < 0.1 | Покой |
| `calm` | 0.1 - 0.3 | Спокойствие |
| `active` | 0.3 - 0.6 | Активность |
| `excited` | 0.6 - 0.8 | Возбуждение |
| `ecstatic` | > 0.8 | Экстаз |
