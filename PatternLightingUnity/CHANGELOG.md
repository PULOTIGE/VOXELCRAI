# Changelog

All notable changes to Pattern Lighting System for Unity 6.

## [1.0.0] - 2024

### Added
- Initial release
- Pattern Light component with 12 built-in patterns
- Custom curve support for patterns
- Pattern Water system with Gerstner waves
- Planar reflections for water
- Foam and caustics effects
- Pattern Shadow system
- Cascaded shadow mapping
- Contact shadows
- PBR shader with pattern emission
- Emissive shader with all patterns
- Water shader with full features
- Pattern Lighting Editor Window
- Custom inspectors for all components
- Live pattern preview
- Material presets
- Sync groups for light synchronization
- Global intensity and speed control
- Flash effect system
- Debug visualization
- Full documentation

### Shaders
- Pattern Lighting/PBR
- Pattern Lighting/Emissive
- Pattern Lighting/Water

### Components
- PatternLight
- PatternWater
- PatternShadow
- PatternShadowCaster
- ContactShadowVolume
- PatternLightingManager

### Editor
- Pattern Lighting Control Panel
- PatternLight custom inspector
- PatternWater custom inspector
- Material creation tools
- Preset system
