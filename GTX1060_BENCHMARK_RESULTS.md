# GTX 1060 4K Benchmark Results

## Обзор

Этот документ содержит результаты бенчмарка для **NVIDIA GTX 1060 (Pascal)** в разрешении 4K.

### Характеристики GTX 1060

- **Архитектура**: Pascal
- **Память**: 6GB GDDR5
- **Memory Bandwidth**: ~192 GB/s
- **CUDA Cores**: 1280
- **TDP**: 120W

---

## Конфигурация теста

- **GPU**: NVIDIA GTX 1060 (Pascal)

## Выводы

### Производительность

✅ **GTX 1060 показывает приемлемую производительность в 4K:**
- 29.63 FPS - играбельно, но не идеально
- Стабильная работа без просадок
- Baked lighting критически важен

### Ограничения

⚠ **GTX 1060 работает на пределе:**
- Высокая утилизация (95.2%)
- Memory bandwidth близко к лимиту
- Старая архитектура (Pascal)

### Рекомендации

1. **Для комфортной игры:**
   - Использовать 1440p вместо 4K
   - Или снизить настройки (Light конфигурация)

2. **Для максимальной производительности:**
   - Обязательно использовать baked lighting
   - Снизить количество частиц и агентов
   - Использовать Sparse/Medium паттерны сцены

3. **Для stress test:**
   - Heavy конфигурация покажет ~20-25 FPS
   - Полезно для тестирования стабильности

## Использование

### Запуск бенчмарка

```bash
# Стандартный бенчмарк
cargo run --bin benchmark-gtx1060

# Легкий бенчмарк
cargo run --bin benchmark-gtx1060 light

# Тяжелый бенчмарк (stress test)
cargo run --bin benchmark-gtx1060 heavy

# Симуляция
cargo run --bin simulate-gtx1060
```
