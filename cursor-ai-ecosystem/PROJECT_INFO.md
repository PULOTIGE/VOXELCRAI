# Cursor AI Ecosystem - Информация о проекте

## Описание

**Cursor AI Ecosystem** - экспериментальный проект по созданию самоорганизующейся экосистемы искусственного интеллекта на основе биологических и когнитивных принципов.

## Ключевые особенности

### 🧬 Нуклеотидная память
- 62 миллиона ячеек памяти (опционально)
- Эпигенетические метки
- Семантические векторы
- Динамическая экспрессия

### 🎮 Воксели-организмы
- Микро-организмы с внутренней структурой (9 КБ)
- Сенсоры, физика, мысли, эмоции, память
- Система "кайфа" на основе энтропии
- Адаптивное обучение

### 💡 Система "кайфа"
- Вычисление через |dS/dt| (производная энтропии)
- Положительное/отрицательное подкрепление
- Адаптация связей память-мысли

### 🔍 Автоматический поиск
- Периодический поиск концептов (19 минут)
- Интеграция новых терминов
- Расширение базы знаний
- DuckDuckGo API

### 🎨 Паттерны освещения
- База данных паттернов (1 КБ каждый)
- Сферические гармоники (9 коэффициентов)
- Поиск похожих паттернов
- Смешивание для итогового освещения

### 📊 Визуализация
- Реальное время (60 FPS)
- Сетка вокселей с цветовой индикацией
- Графики истории (кайф, энтропия, энергия)
- Список найденных концептов

## Производительность

### Рекомендуемые параметры

| Система | Нуклеотиды | Воксели | FPS |
|---------|------------|---------|-----|
| Слабая | 10,000 | 100 | 30-60 |
| Средняя | 100,000 | 400 | 50-60 |
| Мощная | 1,000,000 | 1,000 | 60 |

### Потребление ресурсов

- **Память**: ~256 МБ (100k нуклеотидов + 400 вокселей)
- **CPU**: 1-2 ядра
- **GPU**: не требуется (опционально для будущих версий)

## Использование

### Быстрый старт

```bash
# Установка
pip install -r requirements.txt

# Запуск
python main.py

# С параметрами
python main.py --nucleotides 500000 --voxels 800

# Без визуализации
python main.py --no-visualization
```

### Сборка в EXE

```bash
python scripts/build_exe.py
# Результат: dist/CursorAI.exe (Windows)
```

## Концепции

### Нуклеотид

Базовая ячейка памяти с:
- **Base**: A, T, G, C
- **Epigenetic tags**: метилирование, ацетилирование
- **Quantum noise**: псевдослучайность
- **Histone state**: состояние хроматина
- **Semantic vector**: 39-мерное представление смысла

### Воксель

Микро-организм с внутренней структурой:
- **Sensors**: внешние стимулы (1.5 КБ)
- **Physics**: скорость, энергия, температура (1 КБ)
- **Thoughts**: внутренняя обработка (2 КБ)
- **Emotions**: эмоциональное состояние (2 КБ)
- **Memory**: краткосрочная память (2 КБ)

### Система "кайфа"

```
kaif = |dS/dt|

где S = энтропия эмоционального состояния
```

Высокий кайф → усиление обучения  
Низкий кайф → ослабление связей

### Концепты

Автоматический поиск и интеграция новых терминов:
1. Поиск в DuckDuckGo
2. Извлечение терминов
3. Интеграция в базу знаний
4. Обновление ключевых слов
