# Changelog

## [1.0.0] - 2025-11-30

### Добавлено
- Класс `Nucleotide` - базовая ячейка памяти (256 байт)
  - Эпигенетические метки (метилирование, ацетилирование)
  - Квантовый шум для псевдослучайности
  - Состояние гистонов (хроматин)
  - Семантические векторы (39 float32)
  - Система экспрессии генов

- Класс `Voxel` - микро-организм (9 КБ)
  - 512 байт метаданных
  - 1.5 КБ сенсоры
  - 1 КБ физика
  - 2 КБ мысли
  - 2 КБ эмоции
  - 2 КБ память
  - Система "кайфа" на основе энтропии

- Класс `LightPattern` - паттерны освещения (1 КБ)
  - Прямое и непрямое освещение (32 источника)
  - Коэффициенты сферических гармоник (9 SH)
  - Свойства материала (roughness, metalness, albedo)
  - База данных паттернов с поиском похожих

- Модуль `ConceptSearcher`
  - Поиск концептов через DuckDuckGo
  - Извлечение терминов из текста
  - Автоматическое расширение базы знаний
  - Периодический поиск (19 минут)

- Модуль `ConceptIntegrator`
  - Интеграция найденных концептов
  - Граф связей между концептами
  - Статистика по частоте использования

- Планировщик задач (`Scheduler`, `TickerScheduler`)
  - Асинхронное выполнение задач
  - Поддержка периодических задач
  - Фиксированный ticker (60 FPS)
  - Статистика выполнения

- Визуализация (`EcosystemDisplay`)
  - Отображение состояния вокселей в реальном времени
  - Графики истории (кайф, энтропия, энергия)
  - Статистика экосистемы
  - Список найденных концептов
  - Сетка вокселей с цветовой индикацией

- Главный класс `CursorAIEcosystem`
  - Интеграция всех компонентов
  - Управление жизненным циклом
  - Конфигурируемые параметры
  - Периодические задачи

- Скрипт сборки в EXE
  - Автоматическая сборка через PyInstaller
  - Очистка временных файлов
  - Проверка результата

### Документация
- README.md с полным описанием архитектуры
- QUICKSTART_RU.md с инструкциями по запуску
- config.example.py с примерами конфигурации
- Комментарии в коде

### Особенности
- Система нуклеотидной памяти (до 62 млн ячеек)
- Эмоциональная модель вокселей
- Вычисление "кайфа" как |dS/dt|
- Адаптивное обучение через "кайф"
- Автоматический поиск и интеграция концептов
- Гибридная система освещения с паттернами
- Асинхронная архитектура (asyncio)
- Визуализация в реальном времени (Pygame)

### Технологии
- Python 3.8+
- NumPy для векторных операций
- SciPy для энтропии
- Pygame для визуализации
- asyncio для асинхронности
- BeautifulSoup для парсинга
- PyInstaller для сборки EXE

### Известные ограничения
- Поиск DuckDuckGo может блокироваться при частых запросах
- Производительность зависит от количества нуклеотидов
- Визуализация снижает FPS на слабых системах
- Максимум 62 млн нуклеотидов (ограничение памяти)
