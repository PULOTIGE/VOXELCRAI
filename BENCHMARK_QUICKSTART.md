# Быстрый старт: 4K Benchmark для RTX 4070

## Быстрый запуск

```bash
# Сборка в release режиме (рекомендуется)
cargo build --release

# Запуск стандартного бенчмарка (RTX 4070 target)
cargo run --release --bin benchmark

# Легкий тест
cargo run --release --bin benchmark light

# Тяжелый stress test
cargo run --release --bin benchmark heavy
```

## Через основной движок

```bash
cargo run --release --bin adaptive-entity-engine
```

В окне нажмите:
- **B** - Стандартный 4K бенчмарк
- **L** - Легкий 4K бенчмарк  
- **H** - Тяжелый 4K бенчмарк

## Что тестируется

### Standard Benchmark (RTX 4070 Target)
- **2M частиц** с GPU compute
- **5K агентов** с FSM поведениями
- **Dense сцена** с множеством объектов
- **4K разрешение** (3840x2160)
- **30 секунд** теста

### Ожидаемые результаты для RTX 4070
- Average FPS: **55-65**
- 1% Low FPS: **45-55**
- GPU Utilization: **85-95%**

## Результаты

После завершения бенчмарка в консоли будет выведена статистика:
- Средний/мин/макс FPS
- Время кадра
- 1% и 0.1% Low FPS (стабильность)
- Загрузка GPU
- Использование VRAM

## Примечания

- Для точных результатов используйте `--release` режим
- Убедитесь, что драйверы GPU обновлены
- Закройте фоновые приложения для чистых результатов
- Бенчмарк автоматически завершится после указанного времени
